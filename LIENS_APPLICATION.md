## 🎯 WORKFLOW INSCRIPTION

### Professeurs
1. Inscription: https://school-wheat-six.vercel.app/frontend/inscription-professeur.html
2. Admin valide dans: Dashboard Admin → Gestion des Inscriptions → Professeurs
3. Email automatique envoyé avec identifiants
4. Connexion: https://school-wheat-six.vercel.app/frontend/connexion-professeur.html

### Étudiants
1. Inscription: https://school-wheat-six.vercel.app/frontend/inscription.html
2. Admin valide dans: Dashboard Admin → Gestion des Inscriptions → Étudiants
3. Email automatique envoyé avec identifiants
4. Connexion: https://school-wheat-six.vercel.app/frontend/index.html

### Services Administratifs
1. Inscription via formulaires dédiés (Communication, Comptabilité, Académique)
2. Admin valide dans: Dashboard Admin → Gestion des Inscriptions → [Service]
3. Email automatique envoyé avec identifiants
4. Connexion via page dédiée au service
